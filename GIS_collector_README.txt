GIS Data Collector - README

Saved script: /mnt/data/gis_collector.py
Excel file used: /mnt/data/Biểu mẫu dữ liệu hạ thế GIS_TBA_ĐZ.xlsx

Usage:
  1. Ensure you have Python 3 and dependencies:
     pip install pandas openpyxl

  2. Run the GUI:
     python /mnt/data/gis_collector.py "/mnt/data/Biểu mẫu dữ liệu hạ thế GIS_TBA_ĐZ.xlsx"

Features:
  - Open workbook and choose sheet
  - View first 12 columns of the sheet in a table
  - Select a row and mark/unmark as 'Collected'
  - Save changes back to the Excel workbook or export to CSV
